Temple Raider

A 2D Dungeon Crawler game with procedurally generated dungeons, where the player must look for a treasure inside the dungeon and escape through a portal.

Each location has different variations of terrain and there's two types of enemies that have different behaviour when they get closer to the player.

This project was developed as a part of my University coursework and I decided to make some changes, such as fixing some of the world generation bugs and reworking the AI.

What I Worked on:

  • Worked on the game design.
  • Programmed the procedurally generated dungeons.
  • Programmed the player's controls and movement.
  • Programmed the player's animations.
  • Other minor stuff: camera controller, UI programming.

Project Information:

  • Technologies - Unity, VS Code
  • Languages - C#
  • Project Management - Github

Source code is available on Github.

Game build is available on Itch.io.

Accursed

Accursed is a short 3º person adventure game, where the player must wield the power of the Cursed Orbs and find a way to escape the dungeon.

Each Cursed Orb contains a different power, which will help the player throughout the dungeon, but also inflict a negative effect upon them while holding an Orb. They also work as Keys, being able to fit in certain sockets.

This game was originaly made in a 5-day Game Jam simulation. And later we decided to make some changes, such as fixing some of the game-breaking bugs and completely reworked the level design. Both the original Game Jam version, and the Altered version are available to download and play.

What I Worked on:

  • Worked on the game design.
  • Programmed the player's controls and movement.
  • Programmed the player's animations.
  • Worked on the player interaction with orbs.
  • Programmed the enemies behaviours.
  • Other minor stuff: camera controller, UI programming.
